NEMO.define_field_accessor — Method
define_field_accessor(name, rank, element_type, readwrite)

Generate (and export) the get_$name! accessor — and, when readwrite, the matching set_$name! — that bridges the Julia array to NEMO's nemo_get_$name / nemo_set_$name symbols. rank selects the 2D or 3D interior shape and element_type is the Julia element type the symbol exchanges (Float64 for the C double buffers, Int32 for integer fields).

NEMO.external_forcing_source — Method
external_forcing_source()

Path to the bundled usrdef_sbc.F90 that hands surface-flux control to an external driver; injected into MY_SRC when a configuration is built with external_forcing = true.

NEMO.load_namelists! — Method
load_namelists!(configuration::NemoConfiguration; filenames)

Read every namelist file from the configuration's run directory into the configuration.namelists dictionary, keyed by filename. Files missing from disk are silently skipped.
